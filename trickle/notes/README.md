# Vinit Limkar - AI & Machine Learning Engineer Portfolio

A premium, dark-themed portfolio website showcasing AI/ML projects, skills, and achievements.

## Overview

This is a modern portfolio website for Vinit Limkar, an AI & Machine Learning Engineering student and Smart India Hackathon 2025 Winner. The site features smooth animations, interactive components, and a professional dark theme optimized for tech recruiters.

## Features

- **Dark Theme**: High-contrast design with cyan and pink accents
- **Smooth Animations**: Fade-in, slide-in, and hover effects
- **Responsive Layout**: Mobile-friendly design
- **Interactive Sections**: Hero, About, Skills, Experience, Projects, Achievements, Contact
- **Timeline Experience**: Visual timeline for work experience
- **Project Cards**: Interactive cards with hover effects and external links

## Technology Stack

- React 18
- TailwindCSS
- Lucide Icons
- Babel (for JSX transformation)

## Sections

1. **Hero**: Introduction with gradient text and CTAs
2. **About**: Two-column layout with highlighted keywords
3. **Skills**: 8 categorized skill areas with icons
4. **Experience**: Timeline-style work history
5. **Projects**: Featured AI/ML projects with GitHub links
6. **Achievements**: Major accomplishments and awards
7. **Contact**: Social links and collaboration CTA
